## 2. Scope of Work

The project encompasses the following four core modules:

### A. Core Video & Audio Features (The Foundation)
*   **HD Video Conferencing:** Adaptive video quality that adjusts to user bandwidth (720p/1080p support).
*   **Crystal-Clear Audio:** Integrated echo cancellation and noise suppression for professional audio quality.
*   **Instant Joining:** Seamless join experience via unique meeting links (no aggressive sign-ups required for guests).
*   **Dynamic Grid View:** Support for 2 to 100+ participants with auto-layout adjustments.
*   **Active Speaker Detection:** Visual highlighting of the current speaker.
*   **Media Controls:** Instant Mute/Unmute and Camera On/Off toggles.
*   **Resilience:** Auto-reconnect functionality for network drops and a low-bandwidth fallback mode (audio-only/low-res video).

### B. Meeting Management & Controls
*   **Instant & Scheduled Meetings:** Ability to create immediate rooms or schedule them for later (calendar link generation).
*   **Lobby / Waiting Room:** Security feature allowing hosts to Admit or Deny requested participants.
*   **Host Controls:**
    *   Remove participants.
    *   Mute all / individual participants.
    *   Lock meeting (prevent new joins).
    *   End meeting for all.
*   **Roles:** Distinct privileges for Host, Co-host, and Attendees.
*   **Interaction:** "Raise Hand" feature for orderly discussions.
*   **Participant List:** Real-time list showing status (muted/video off/active).

### C. Screen Sharing & Presentation
*   **Flexible Sharing:** Options to share the entire screen, a specific application window, or a browser tab.
*   **Presenter Mode:** Seamless switching between presenters without disconnecting.
*   **Simultaneous View:** Participants can view the presentation and the presenter's video feed side-by-side.
*   **Optimization:** Specific modes for text readability vs. video frame rate.
*   **Pause/Stop:** Instant controls to pause or stop sharing privacy.

### D. Chat & Collaboration
*   **In-Meeting Chat:** Real-time text messaging pane for all participants.
*   **Context:** Chat timestamps and integration of emoji reactions (👍, 👏, 😂, etc.).
*   **File Sharing:** Integration for sharing documents/files directly in chat (or via cloud link previews).
*   **Rich Links:** Automatic generation of link previews.
*   **Moderation:** Host capability to clear chat or moderate messages if required.
